# Альтернативные тренировки по типам пользователей

Все упражнения — из базы приложения (exercises_255.json). Термины: **суперсет** — два упражнения подряд без отдыха, отдых после пары; **круговая** — все упражнения по одному подходу друг за другом, отдых после полного круга.

Колонки: **ID** — идентификатор упражнения в базе; **Мышцы** — основная группа, в скобках — второстепенно участвующие. Уровневые версии в строке относятся к тем же мышечным группам, что и базовое упражнение первой колонки.

Обозначения прогрессии: «→» — шаг вперёд при выполнении верхней планки повторов во всех подходах; «↩» — шаг назад при провале (не добил нижнюю планку в двух тренировках подряд).

## 2. Опытный любитель (1–5+ лет)
